# desearch
A wrapper around [sci-hub.tw](sci-hub.tw) for downloading Research Papers.

## Installation
```shell
git clone https://github.com/krAshwin/desearch.git
cd desearch 
chmod +x desearch
```
## Usage
```shell
./desearch {url}
```

## Known url sources
- IEEE
- ACM
- Science Direct
- Springer
